# TaskFlow Backend

A Spring Boot based Task Management API.

## Features

- User CRUD APIs
- Task CRUD APIs
- DTO-based architecture
- Custom ID generation
- Request validation using Jakarta Validation
- Global exception handling
- Custom enum validation responses
- One-to-Many relationship between User and Task
- JPA entity mapping
- Spring Data JPA
- MySQL Database Integration

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Lombok
- Maven
